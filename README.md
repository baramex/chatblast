# installation
C'est un projet node, il suffit de clone le projet puis de lancer l'app.js:
```console
git clone git@github.com:baramex/chatbox.git
npm i
node app.js
```
Ensuite, rendez-vous sur votre navigateur et entrer `localhost:1500` et voilà tout !
